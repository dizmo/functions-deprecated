# @dizmo/functions-deprecated

A decorator to deprecate class methods.

## Usage

### Installation

```sh
npm install @dizmo/functions-deprecated --save
```

### Require

```javascript
import '@dizmo/functions-deprecated';
```

### Example(s)

```javascript
class MyClass {
    @deprecated('message')
    method() {
        ...
    }
}
```
